# Лапа помощи

1. В корне репозитория выполните:

```bash
npm install
```

2. Создайте файл `.env.local` в корне проекта

```env
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000
```

3. Запустите бэкенд на порту **8000** 

4. Затем в терминале введите:

```bash
npm run dev
```

Сайт доступен по адресу http://localhost:3000
